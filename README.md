# 🚀 Responsive Landing Page – CyberShield Academy

## 📌 Task Description

**Task:** Build a Responsive Landing Page  

Create a responsive landing page for a fictional product or service. The page should demonstrate modern UI/UX design, responsiveness across devices, and interactive elements using HTML, CSS, and JavaScript.

---

## 💡 Project Overview

This project is a **responsive landing page** for a fictional cybersecurity learning platform called **CyberShield Academy**.

The website promotes an online course platform that teaches:
- Ethical Hacking  
- Penetration Testing  
- Cybersecurity Fundamentals  

---

## 🛠️ Technologies Used

- HTML5 – Structure and layout  
- Tailwind CSS – Styling and responsiveness  
- Custom CSS – Animations and UI effects  
- JavaScript (Vanilla JS) – Interactivity  

---

## ✨ Features Implemented

### 🎨 Responsive Design
- Works on mobile, tablet, and desktop  
- Flexible grid layouts using Tailwind CSS  
- Mobile-friendly navigation menu  

### 🧭 Navigation
- Sticky navbar with scroll effect  
- Smooth scrolling to sections  

### 📢 Sections Included
- Hero Section (Introduction + CTA)
- Features Section
- Curriculum Section
- Testimonials Section
- Pricing Section
- Footer

### ⚙️ Interactivity
- Mobile menu toggle  
- Modal popups (Login / Enroll / Contact)  
- Toast notifications  
- Scroll-based animations
